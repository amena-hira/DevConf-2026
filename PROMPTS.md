Create a clean and creative **Sponsors section** for the existing **DevConf 2026** website using only basic HTML and CSS.

The design should match the current page:

* Modern conference style
* White background
* Bold black headings
* Dark navy details
* Sponsor logos
* Royal-blue buttons

## Content

Include:

* Small centered text: **POWERING DEVCONF 2026**
* Main heading: **Meet Our Sponsors**
* Short description thanking the companies supporting the conference.

## Sponsor Layout

### Platinum Partners

* Two large sponsor cards per row on desktop.
* Each card includes:

  * Company logo (using a publicly accessible logo image URL)
  * Company name
  * Short description
  * "Platinum Partner" label

### Gold Sponsors

* Three sponsor cards per row.
* Medium-sized cards including:

  * Company logo
  * Company name

### Community Sponsors

* Four small logo cards per row.
* Minimal design with company logos.

Use recognizable technology companies such as Google, Microsoft, GitHub, Intel, NVIDIA, Docker, Vercel, Cloudflare, DigitalOcean, MongoDB, Atlassian, or similar. Use their official logo image URLs or publicly accessible logo URLs.

## Bottom Call-to-Action

Create a dark navy sponsorship banner containing:

* Heading: **Put Your Brand at the Heart of DevConf**
* Short supporting text
* Royal-blue button: **Become a Sponsor**

## CSS Requirements

Use simple and readable CSS only.

Allowed CSS:

* display: flex
* display: grid
* grid-template-columns
* gap
* margin
* padding
* width
* max-width
* background
* color
* border
* border-radius
* box-shadow
* font-size
* font-weight
* text-align
* align-items
* justify-content
* object-fit

Do not use:

* CSS variables
* Pseudo-elements (::before, ::after)
* Complex selectors
* CSS animations
* JavaScript
* Bootstrap
* Tailwind

Return one self-contained HTML snippet with a `<style>` block only. Do not include `<html>`, `<head>`, or `<body>` tags. The section should be responsive and easy for beginners to edit and paste directly below the Speakers section.
